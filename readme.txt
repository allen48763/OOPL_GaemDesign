人物==>

2022/04/28
人物動作: 靜置，左/右前進，左/右趴下，左/右跳躍，左/右滾動
動作均有移動動畫
當人物往左，地圖不會移動
當人物往右，如果未超過 座標K (大概是畫面座標的1/3)，地圖不會移動
超過則地圖跟著移動
人物能發射子彈，目前飛射的方向有: 左、右、上
預計子彈可顯示動畫 
人物有死亡動畫
地圖左上有設置HP: 5，但目前還沒實裝 "觸發死亡的事件處理"
當人物移動到地圖的最右邊界外會閃退 (因為超出地圖本身的陣列，待調整)

2022/05/12
新增了當玩家死亡次數到達五次時，切換場景到遊戲Over的State
新增了玩家重生後的短暫無敵
